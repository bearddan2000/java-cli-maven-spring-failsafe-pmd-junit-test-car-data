# java-cli-maven-spring-failsafe-pmd-junit-test-car-data

## Description
A POC for sprimg app using JUnit
framework with failsafe and
pmd plugins.

## Tech stack
- java
- maven
  - spring
  - junit
  - failsafe
  - pmd

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- pmd report found at bin/target/site

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
